# Loan Risk Predictor

A machine learning web app that estimates the probability a loan applicant
will default, based on their financial profile. Built with scikit-learn /
XGBoost and served through a Streamlit UI.

## Features

- **Predict** — enter an applicant's profile and get a risk level (Low /
  Medium / High), a default probability, and the top factors driving the
  prediction.
- **Model Info** — compares Logistic Regression, Random Forest, and XGBoost
  on accuracy, ROC-AUC, precision, recall, and F1; shows ROC curves and
  feature importance for the selected best model.
- **About** — project and dataset background.

## Tech stack

Python, scikit-learn, XGBoost, pandas/NumPy, Streamlit, Matplotlib/Seaborn,
joblib, Docker (optional).

## Dataset

Based on the ["Give Me Some Credit"](https://www.kaggle.com/c/GiveMeSomeCredit/data)
Kaggle dataset. To use the real data, download `cs-training.csv` from Kaggle
and place it at `data/cs-training.csv` before running `model/train.py`. If
that file isn't present, the training script automatically generates a
synthetic dataset with the same features and realistic, logically consistent
risk relationships (see `generate_synthetic_data` in `model/train.py`).

## Folder structure

```
loan-risk-predictor/
  app.py                   # Streamlit entry point
  model/
    train.py                # data cleaning, feature engineering, training, evaluation
    predict.py               # loads the saved model and scores new applicants
    loan_model.joblib        # saved best-model bundle (generated by train.py)
  data/
    cs-training.csv          # optional: real Kaggle dataset (not included)
    sample_data.csv          # dataset actually used for training (generated)
  requirements.txt
  Dockerfile
  README.md
```

## Run locally

```bash
python -m venv .venv
source .venv/bin/activate        # Windows: .venv\Scripts\activate

pip install -r requirements.txt

# Train the model (creates model/loan_model.joblib and data/sample_data.csv)
python model/train.py

# Launch the app
streamlit run app.py
```

Then open the URL Streamlit prints (usually http://localhost:8501).

## Deploy to Hugging Face Spaces

1. Create a new Space at [huggingface.co/new-space](https://huggingface.co/new-space)
   and choose the **Streamlit** SDK.
2. Push this repo's contents to the Space's git remote (or upload the files
   via the web UI): `app.py`, `model/`, `data/`, `requirements.txt`.
3. Make sure `model/loan_model.joblib` is committed (or add a build step that
   runs `python model/train.py` before the app starts) so the app has a
   trained model to load.
4. The Space will install `requirements.txt` and run `app.py` automatically —
   no extra configuration needed.

## Run with Docker

```bash
docker build -t loan-risk-predictor .
docker run -p 8501:8501 loan-risk-predictor
```
